# Exposeduc

Este é um projeto [Next.js](https://nextjs.org) inicializado com [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Finalidade do Sistema

O **Exposeduc** é uma plataforma moderna desenvolvida para facilitar a gestão e a exposição de conteúdos educacionais. Construído com tecnologias de ponta como Next.js, Prisma e Tailwind CSS, o sistema visa proporcionar uma experiência de usuário fluida, rápida e responsiva, conectando educadores e alunos de maneira eficiente e interativa.

## Começando

Primeiro, execute o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

Você pode começar a editar a página modificando `src/app/page.tsx` (ou `app/page.tsx` dependendo da estrutura). A página é atualizada automaticamente conforme você edita o arquivo.

Este projeto usa [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) para otimizar e carregar automaticamente a fonte [Geist](https://vercel.com/font), uma nova família de fontes da Vercel.
